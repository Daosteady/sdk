# Steady USDX

**Steady USDX** is a mortgage-backed stablecoin designed to deliver sustainable yield sourced from real-world U.S. real estate debt.

## Overview

Steady USDX bridges decentralized finance (DeFi) with real-world assets (RWA) by backing its value with U.S. mortgage debt. The protocol focuses on stability, transparency, and consistent yield generation.

## How It Works

1. Capital is allocated to U.S. real estate mortgage debt.
2. Mortgage interest generates real-world yield.
3. Yield is distributed to USDX holders.
4. Stablecoin value remains supported by underlying assets.

## Use Cases

- Stable yield earning
- DeFi collateral
- Treasury management
- Hedge against crypto volatility
- Exposure to U.S. real estate debt

## Philosophy

Steady is built on the belief that sustainable DeFi must be backed by real economic activity. By anchoring value to real-world assets, Steady USDX aims to create long-term stability and trust.
